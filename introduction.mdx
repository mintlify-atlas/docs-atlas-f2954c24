  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-4">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your full-stack application running locally in just a few steps
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the project to your local machine:
      
      ```bash
      git clone https://github.com/challa-harini/mini-project.git
      cd mini-project
      ```
    </Step>

    <Step title="Install dependencies">
      Install dependencies for both the client and server:
      
      <Tabs>
        <Tab title="Client">
          ```bash
          cd client
          npm install
          ```
        </Tab>
        <Tab title="Server">
          ```bash
          cd server
          npm install
          ```
        </Tab>
      </Tabs>
    </Step>

    <Step title="Start the backend server">
      Run the Express server on port 5000:
      
      ```bash
      cd server
      node index.js
      ```
      
      You should see:
      ```
      ✅ Server running on port 5000
      ```
    </Step>

    <Step title="Start the React frontend">
      In a new terminal, start the React development server:
      
      ```bash
      cd client
      npm start
      ```
      
      Your browser will open to `http://localhost:3000` showing the message from your Express backend.
    </Step>
  </Steps>
</div>
