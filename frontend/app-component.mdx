## Overview

The `App` component is the root component of the React application. It demonstrates a simple client-server communication pattern by fetching data from the backend API.

<Info>
This component is located at `client/src/App.js`
</Info>

## Complete Source Code

```jsx App.js
import { useEffect, useState } from "react";

function App() {
  const [message, setMessage] = useState("");

  useEffect(() => {
    fetch("http://localhost:5000/")
      .then((res) => res.text())
      .then((data) => setMessage(data))
      .catch((err) => console.error(err));
  }, []);

  return (
    <div style={{ textAlign: "center", marginTop: "50px" }}>
      <h1>Frontend Connected to Backend</h1>
      <h2>{message}</h2>
    </div>
  );
}

export default App;
```

## Key Concepts

<AccordionGroup>
  <Accordion title="Why useEffect?">
    The `useEffect` hook is used because:
    
    - Fetch operations are **side effects** (they interact with external systems)
    - We need to run the fetch **after** the component is mounted to the DOM
    - It prevents the fetch from running on every render
    
    Without `useEffect`, attempting to fetch during render would cause issues:
    ```jsx
    // ❌ DON'T DO THIS
    function App() {
      const [message, setMessage] = useState("");
      
      // This would run on every render, causing infinite loop!
      fetch("http://localhost:5000/")
        .then((res) => res.text())
        .then((data) => setMessage(data));  // Triggers re-render
      
      return <div>{message}</div>;
    }
    ```
  </Accordion>
  
  <Accordion title="Dependency Array">
    The empty dependency array `[]` is crucial:
    
    ```jsx
    useEffect(() => {
      // This runs ONCE after initial render
    }, []);  // Empty array = run once
    ```
    
    Different behaviors based on the dependency array:
    
    | Dependency Array | Behavior |
    |-----------------|----------|
    | `[]` | Runs once after mount |
    | `[value]` | Runs when `value` changes |
    | No array | Runs after every render |
  </Accordion>
  
  <Accordion title="Error Handling">
    The `.catch()` ensures errors don't crash the app:
    
    ```jsx
    .catch((err) => console.error(err));
    ```
    
    Common errors that might occur:
    
    - **Network Error:** Backend server is not running
    - **CORS Error:** Backend doesn't allow requests from `localhost:3000`
    - **Timeout:** Backend takes too long to respond
    
    <Tip>
    For production apps, consider showing error messages to users instead of just logging to console.
    </Tip>
  </Accordion>
  
  <Accordion title="Response Format">
    The code uses `.text()` to parse the response:
    
    ```jsx
    .then((res) => res.text())
    ```
    
    For different response types, use:
    
    | Method | Use Case | Example |
    |--------|----------|----------|
    | `.text()` | Plain text | `"Hello World"` |
    | `.json()` | JSON data | `{"message": "Hello"}` |
    | `.blob()` | Binary data | Images, files |
    | `.arrayBuffer()` | Raw binary | Audio, video |
  </Accordion>
</AccordionGroup>

## Customization Ideas

<CardGroup cols={2}>
  <Card title="Loading State" icon="spinner">
    Add a loading indicator while fetching:
    
    ```jsx
    const [loading, setLoading] = useState(true);
    
    useEffect(() => {
      fetch("http://localhost:5000/")
        .then((res) => res.text())
        .then((data) => {
          setMessage(data);
          setLoading(false);
        });
    }, []);
    
    if (loading) return <div>Loading...</div>;
    ```
  </Card>
  
  <Card title="Error UI" icon="triangle-exclamation">
    Display errors to users:
    
    ```jsx
    const [error, setError] = useState(null);
    
    useEffect(() => {
      fetch("http://localhost:5000/")
        .then((res) => res.text())
        .then((data) => setMessage(data))
        .catch((err) => setError(err.message));
    }, []);
    
    if (error) return <div>Error: {error}</div>;
    ```
  </Card>
  
  <Card title="JSON API" icon="code">
    Switch to JSON responses:
    
    ```jsx
    fetch("http://localhost:5000/api/data")
      .then((res) => res.json())
      .then((data) => setMessage(data.message));
    ```
  </Card>
  
  <Card title="Refresh Button" icon="rotate">
    Add manual refresh capability:
    
    ```jsx
    const fetchData = () => {
      fetch("http://localhost:5000/")
        .then((res) => res.text())
        .then((data) => setMessage(data));
    };
    
    useEffect(fetchData, []);
    
    return (
      <div>
        <h2>{message}</h2>
        <button onClick={fetchData}>Refresh</button>
      </div>
    );
    ```
  </Card>
