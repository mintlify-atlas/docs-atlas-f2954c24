## Global Styles (index.css)

The `index.css` file defines base styles for the entire application:

```css index.css
body {
  margin: 0;
  font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto', 'Oxygen',
    'Ubuntu', 'Cantarell', 'Fira Sans', 'Droid Sans', 'Helvetica Neue',
    sans-serif;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}

code {
  font-family: source-code-pro, Menlo, Monaco, Consolas, 'Courier New',
    monospace;
}
```

## Component Styles (App.css)

The `App.css` file provides default styles that can be used for App component variants:

```css App.css
.App {
  text-align: center;
}

.App-logo {
  height: 40vmin;
  pointer-events: none;
}

@media (prefers-reduced-motion: no-preference) {
  .App-logo {
    animation: App-logo-spin infinite 20s linear;
  }
}

.App-header {
  background-color: #282c34;
  min-height: 100vh;
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  font-size: calc(10px + 2vmin);
  color: white;
}

.App-link {
  color: #61dafb;
}

@keyframes App-logo-spin {
  from {
    transform: rotate(0deg);
  }
  to {
    transform: rotate(360deg);
  }
}
```

<Note>
These styles are from Create React App's default template. The current App component uses inline styles instead, so these classes are available but not actively used.
</Note>

## Current Component Styling

The current `App` component uses **inline styles** instead of CSS classes:

```jsx App.js
<div style={{ textAlign: "center", marginTop: "50px" }}>
  <h1>Frontend Connected to Backend</h1>
  <h2>{message}</h2>
</div>
```

<Tabs>
  <Tab title="Inline Styles">
    ### Current Approach
    
    ```jsx
    style={{ textAlign: "center", marginTop: "50px" }}
    ```
    
    **Advantages:**
    - Quick and simple for small components
    - Styles are scoped to the component
    - No additional CSS classes needed
    
    **Disadvantages:**
    - Harder to reuse across components
    - No pseudo-classes (`:hover`, `:focus`, etc.)
    - Slightly less performant than CSS classes
  </Tab>
  
  <Tab title="Using CSS Classes">
    ### Alternative: CSS Classes
    
    You can switch to using the available CSS classes:
    
    ```jsx
    // Import the CSS file
    import './App.css';
    
    function App() {
      return (
        <div className="App">
          <h1>Frontend Connected to Backend</h1>
          <h2>{message}</h2>
        </div>
      );
    }
    ```
    
    Then customize `App.css`:
    ```css
    .App {
      text-align: center;
      margin-top: 50px;
    }
    ```
  </Tab>
  
  <Tab title="Using App-header">
    ### Alternative: Full-Screen Layout
    
    Use the `.App-header` class for a full-screen dark theme:
    
    ```jsx
    import './App.css';
    
    function App() {
      return (
        <div className="App-header">
          <h1>Frontend Connected to Backend</h1>
          <h2>{message}</h2>
        </div>
      );
    }
    ```

## Best Practices

<CardGroup cols={2}>
  <Card title="Use CSS Variables" icon="palette">
    Define reusable color schemes:
    
    ```css
    :root {
      --primary-color: #4a90e2;
      --secondary-color: #2ecc71;
      --background: #ffffff;
      --text: #333333;
    }
    
    .App h1 {
      color: var(--primary-color);
    }
    ```
  </Card>
  
  <Card title="Mobile-First Design" icon="mobile">
    Start with mobile styles, then add desktop:
    
    ```css
    /* Mobile styles */
    .App {
      padding: 1rem;
    }
    
    /* Desktop styles */
    @media (min-width: 768px) {
      .App {
        padding: 2rem;
      }
    }
    ```
  </Card>
  
  <Card title="Semantic Class Names" icon="tag">
    Use meaningful, descriptive names:
    
    ```css
    /* Good */
    .message-container { }
    .primary-heading { }
    
    /* Avoid */
    .box1 { }
    .blue-text { }
    ```
  </Card>
  
  <Card title="Accessibility" icon="universal-access">
    Ensure sufficient contrast and focus states:
    
    ```css
    .App button:focus {
      outline: 2px solid #4a90e2;
      outline-offset: 2px;
    }
    
    /* Minimum contrast ratio: 4.5:1 */
    .App h1 {
      color: #1a1a1a; /* On white background */
    }
    ```
  </Card>
</CardGroup>
